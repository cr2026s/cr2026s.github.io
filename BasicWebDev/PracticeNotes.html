Lists
  1. start with ol for ordered list, or ul for unordered list
  2. then next to the ol, but a type attribute (type="") 1 for numbers A/a for letters, or I/i for roman numerals
  3. then put the start attribut for what number you'll start the list at (usually 1) only works with numbers
  4. if you want the list to be a countdown, create a reversed attribute, and set it to reversed, only works with numbers
  5. start each line in the list with li in anckle brackets
  6. don't forget closing tags!
